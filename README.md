## 最新功能

- 自定义后台拖拽界页面；
- 默认3种商品分类页面；
- 订单管理；
- 退款退货管理；
- 收藏列表；
- 地址管理，1.用户添加，2.调用微信地址管理接口；
- 购物车管理，支持切换规格；
- 商品管理，支持多规格切换
